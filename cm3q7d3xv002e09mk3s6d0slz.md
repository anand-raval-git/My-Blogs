### 21\. **What does AWS Elastic Load Balancer (ELB) do?**

a) Stores objects in the cloud  
b) Distributes incoming application traffic across multiple targets  
c) Monitors application performance  
d) Provides CDN functionality  
**Answer:** b) Distributes incoming application traffic across multiple targets  
**Explanation:** ELB ensures fault tolerance by distributing traffic to healthy instances.

---

### 22\. **Which AWS service allows you to run containerized applications without managing servers?**

a) Amazon ECS  
b) AWS Fargate  
c) Amazon EC2  
d) AWS Lambda  
**Answer:** b) AWS Fargate  
**Explanation:** Fargate provides a serverless compute engine for containers, eliminating server management.

---

### 23\. **What is the benefit of using Amazon CloudFront?**

a) Reduces storage costs  
b) Improves security of the data  
c) Delivers content with low latency  
d) Provides backup for on-premises data  
**Answer:** c) Delivers content with low latency  
**Explanation:** CloudFront uses edge locations to cache content closer to users for faster delivery.

---

### 24\. **Which service is best for automating application deployment?**

a) AWS CodeBuild  
b) AWS CodeDeploy  
c) AWS CodePipeline  
d) AWS CloudFormation  
**Answer:** b) AWS CodeDeploy  
**Explanation:** AWS CodeDeploy automates application deployments to various environments like EC2 or Lambda.

---

### 25\. **What does an Amazon S3 bucket policy allow you to do?**

a) Attach an IAM role to a bucket  
b) Manage lifecycle policies for bucket objects  
c) Grant permissions to objects stored in the bucket  
d) Configure bucket encryption  
**Answer:** c) Grant permissions to objects stored in the bucket  
**Explanation:** Bucket policies control access to objects in an S3 bucket at a bucket level.

---

### 26\. **Which AWS tool helps you view, monitor, and budget your AWS spending?**

a) AWS CloudTrail  
b) AWS Cost Explorer  
c) AWS CloudWatch  
d) AWS Pricing Calculator  
**Answer:** b) AWS Cost Explorer  
**Explanation:** Cost Explorer provides detailed insights into your AWS usage and costs.

---

### 27\. **What does Amazon RDS automatically handle?**

a) Application performance tuning  
b) Database backups and maintenance tasks  
c) Network traffic distribution  
d) Encryption of all application data  
**Answer:** b) Database backups and maintenance tasks  
**Explanation:** RDS simplifies database management by automating tasks like backups and patching.

---

### 28\. **What is the primary function of AWS CloudTrail?**

a) Real-time monitoring of applications  
b) Logging API calls and user activities  
c) Automating cloud infrastructure  
d) Encrypting data in transit  
**Answer:** b) Logging API calls and user activities  
**Explanation:** CloudTrail tracks user activity and API calls for auditing and compliance.

---

### 29\. **Which AWS storage option is designed for high-performance applications requiring low latency?**

a) Amazon S3  
b) Amazon Glacier  
c) Amazon EBS  
d) Amazon EFS  
**Answer:** c) Amazon EBS  
**Explanation:** EBS provides block storage for low-latency access and high performance.

---

### 30\. **What is the role of an Amazon IAM policy?**

a) Define DNS routing rules  
b) Grant or deny permissions to AWS services  
c) Set resource scaling policies  
d) Enable network security  
**Answer:** b) Grant or deny permissions to AWS services  
**Explanation:** IAM policies define what actions are allowed or denied for users, groups, or roles.

### 31\. **Which AWS feature allows you to recover quickly from a disaster?**

a) Cross-Region Replication  
b) AWS Shield  
c) Amazon GuardDuty  
d) AWS Elastic Beanstalk  
**Answer:** a) Cross-Region Replication  
**Explanation:** Cross-Region Replication ensures data redundancy across regions for disaster recovery.

---

### 32\. **Which service provides server-side encryption for data in transit?**

a) AWS CloudHSM  
b) AWS KMS  
c) Amazon S3 with SSL/TLS  
d) AWS Secrets Manager  
**Answer:** c) Amazon S3 with SSL/TLS  
**Explanation:** SSL/TLS encrypts data as it travels between the client and S3.

### 33\. **What does the Amazon EC2 Auto Recovery feature do?**

a) Automatically scales instances based on demand  
b) Restarts instances if a system status check fails  
c) Deletes unhealthy instances  
d) Migrates instances to another region  
**Answer:** b) Restarts instances if a system status check fails  
**Explanation:** Auto Recovery ensures high availability by recovering instances when system issues occur.

### 34\. **Which AWS support plan includes a dedicated Technical Account Manager (TAM)?**

a) Basic  
b) Developer  
c) Business  
d) Enterprise

### 35\. **Which feature of AWS provides detailed billing reports?**

a) AWS Budgets  
b) AWS Cost and Usage Reports

### 36\. **What is the function of an AWS Security Group?**

a) Encrypts data at rest  
b) Acts as a virtual firewall for EC2 instances  
c) Monitors compliance of resources  
d) Manages user access to the AWS Management Console  
**Answer:** b) Acts as a virtual firewall for EC2 instances  
**Explanation:** Security Groups control inbound and outbound traffic to EC2 instances.

### 37\. **What is a key use case for Amazon DynamoDB?**

a) Relational data storage  
b) Analytics on large datasets  
c) NoSQL database for scalable, low-latency applications

**Explanation:** DynamoDB is ideal for high-performance, low-latency NoSQL use cases.
