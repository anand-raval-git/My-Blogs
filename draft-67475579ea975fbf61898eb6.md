### 81\. **Which AWS service enables DNS management?**

a) Amazon Route 53  
b) AWS CloudFormation  
c) AWS CloudTrail  
d) Amazon VPC  
**Answer:** a) Amazon Route 53  
**Explanation:** Route 53 is a scalable Domain Name System (DNS) service for routing end users to internet applications.

---

### 82\. **What does the AWS Well-Architected Framework provide?**

a) Security monitoring tools  
b) Best practices for building secure and efficient cloud architectures  
c) Cost management strategies  
d) Pre-configured resource templates  
**Answer:** b) Best practices for building secure and efficient cloud architectures  
**Explanation:** The Well-Architected Framework guides customers on designing cloud workloads that align with best practices.

---

### 83\. **Which AWS service allows developers to run functions without managing servers?**

a) AWS Batch  
b) AWS Lambda  
c) Amazon ECS  
d) Amazon EMR  
**Answer:** b) AWS Lambda  
**Explanation:** Lambda is a serverless compute service that runs functions in response to events.

---

### 84\. **Which storage class in Amazon S3 is most cost-effective for infrequently accessed data?**

a) S3 Standard  
b) S3 Intelligent-Tiering  
c) S3 Standard-IA  
d) S3 One Zone-IA  
**Answer:** c) S3 Standard-IA  
**Explanation:** S3 Standard-IA (Infrequent Access) is optimized for data that is accessed less frequently but requires quick access.

---

### 85\. **What does the "Shared Responsibility Model" in AWS mean?**

a) AWS handles all aspects of security.  
b) Customers are responsible for physical security.  
c) AWS manages security of the cloud, and customers manage security in the cloud.  
d) Customers and AWS share all security tasks equally.  
**Answer:** c) AWS manages security of the cloud, and customers manage security in the cloud.  
**Explanation:** AWS secures the infrastructure, while customers secure their data and configurations.

a) Amazon SageMaker  
b) AWS Glue  
c) Amazon EMR  
d) AWS Step Functions  
**Answer:** a) Amazon SageMaker

### 87\. **Which AWS service helps you track API usage and calls?**

a) AWS CloudTrail  
b) Amazon CloudWatch  
c) AWS Config  
d) AWS X-Ray  
**Answer:** a) AWS CloudTrail

### 88\. **What is the default maximum retention period for Amazon SQS messages?**

a) 1 day  
b) 4 days  
c) 14 days  
d) 30 days  
**Answer:** c) 14 days  
**Explanation:** SQS messages can be retained for up to 14 days by default.

### 89\. **What does Amazon Elastic File System (EFS) provide?**

a) Relational database storage  
b) Managed file storage for EC2 instances  
c) Backup for S3 data  
d) Object storage for unstructured data  
**Answer:** b) Managed file storage for EC2 instances  
**Explanation:** EFS is a scalable file system for use with AWS compute services.

a) Amazon RDS  
b) Amazon Redshift  
c) Amazon DynamoDB  
d) AWS DataSync  
**Answer:** b) Amazon Redshift

### 91\. **Which service helps manage the lifecycle of EC2 instances using predefined schedules?**

a) AWS Config  
b) AWS Systems Manager  
c) AWS Auto Scaling  
d) AWS OpsWorks  
**Answer:** b) AWS Systems Manager  
**Explanation:** Systems Manager provides features like maintenance schedules and automation for EC2 instances.

### 92\. **What type of traffic does a security group control?**

a) Data replication traffic  
b) Inbound and outbound traffic for instances  
c) Internet routing traffic  
d) VPN traffic only  
**Answer:** b) Inbound and outbound traffic for instances  
**Explanation:** Security groups act as virtual firewalls for EC2 instances, controlling traffic in and out.

### 93\. **Which AWS service is used to collect and visualize operational data?**

a) AWS Config  
b) Amazon CloudWatch  
c) AWS CloudTrail  
d) AWS Budgets  
**Answer:** b) Amazon CloudWatch  
**Explanation:** CloudWatch monitors and visualizes metrics, logs, and alarms for AWS resources.

### 94\. **Which service is designed to accelerate content delivery for dynamic and static content?**

a) Amazon CloudFront  
b) Amazon RDS  
c) Amazon SQS  
d) AWS Glue  
**Answer:** a) Amazon CloudFront  
**Explanation:** CloudFront is a CDN that speeds up content delivery to users globally.

### 95\. **Which AWS service simplifies transferring petabytes of data into AWS?**

a) AWS DataSync  
b) AWS Snowball  
c) AWS Storage Gateway  
d) Amazon S3 Transfer Acceleration  
**Answer:** b) AWS Snowball  
**Explanation:** Snowball is a physical device used for transferring large data volumes securely to AWS.

### 96\. **What is the purpose of Amazon RDS Multi-AZ deployments?**

a) To improve read performance  
b) To provide high availability and failover support  
c) To optimize costs for test environments  
d) To scale horizontally for database writes  
**Answer:** b) To provide high availability and failover support  
**Explanation:** Multi-AZ deployments replicate data to standby instances in different Availability Zones.

### 97\. **Which service provides access control for AWS resources?**

a) AWS IAM  
b) AWS Artifact  
c) AWS Shield  
d) Amazon VPC  
**Answer:** a) AWS IAM  
**Explanation:** IAM allows you to manage access permissions to AWS resources.

### 98\. **What is the primary use of Amazon EBS?**

a) Object storage  
b) Block storage for EC2 instances  
c) File storage  
d) Database storage  
**Answer:** b) Block storage for EC2 instances  
**Explanation:** EBS provides persistent block storage for EC2 instances.

---

### 99\. **Which service provides automated software deployments to EC2 instances?**

a) AWS CodePipeline  
b) AWS CodeDeploy  
c) AWS CloudFormation  
d) AWS Elastic Beanstalk

**Explanation:** CodeDeploy automates the deployment process for applications on EC2 and other compute services.

a) AWS Config  
b) AWS CloudTrail
