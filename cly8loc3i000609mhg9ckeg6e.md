### • list the sub directories and files avaiable in the present directory

```bash
ls
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720177658622/049bab29-2766-4100-b2a1-bdef74e17b80.png align="center")

### • list the files and directories in long list format with extra information

```bash
ls -l
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720177705576/33e257b5-535e-4eda-b1ed-5b7efaa84aa6.png align="center")

### • list all including hidden files and directory

```bash
ls -a
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720177754765/05c23320-d524-49fd-a6a8-2d538f044931.png align="center")

### • list all the files having .sh extension.

```bash
ls *.sh
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720177839145/d70f64dd-06b0-45f0-9f29-0f6429e8ceb3.png align="center")

> Note :- whenever you want somethign specific file use \* and then write format of file

### • list the files and directories with index numbers inodes

```bash
ls -i
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720177946855/9b6410d1-1a72-405a-aae5-36d1cdfe8fa3.png align="center")

### • list only directories.(we can also specify a pattern)

```bash
 ls -d */
```
