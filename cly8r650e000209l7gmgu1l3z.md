## DAY 3 :- TASK

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720185040551/60247b6d-6867-4eec-85aa-f1a9936776f2.png align="center")

## 1\. View the content of a file and display line numbers.

```bash
cat -n #filename
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720185321696/5377f205-2e6d-4290-a2a9-5db24354a750.png align="center")

## 2\. Change the access permissions of files to make them readable, writable, and executable by the owner only.

```bash
chmod 777 #filename
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720185480050/f7a0789a-dfeb-4584-9150-d63178db3615.png align="center")

## 3\. Check the last 10 commands you have run.

```bash
history | tail -10
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720185610208/72124243-e60d-4b0a-8283-de2ee3b89d64.png align="center")

## 4\. Remove a directory and all its contents.

```bash
rm -r #directoryname
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720185727371/66bcb87a-ac02-41cf-948e-86ba4995818e.png align="center")

## 5\. Create a `fruits.txt` file, add content (one fruit per line), and display the content.

```bash
touch fruits.txt #creating file
vim fruits.txt #editing file as one fruite per line
cat fruits.txt #display the content
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720185932644/83b3944c-b79a-4510-895f-00456c7ab98c.png align="center")

## 6\. Add content in `devops.txt` (one in each line) - Apple, Mango, Banana, Cherry, Kiwi, Orange, Guava. Then, append "Pineapple" to the end of the file.

```bash
vim fruits.txt #editing file as one fruite per line
cat fruits.txt #display the content
echo "pineapple" >> fruits.txt #adding pineapple in last of fruits.txt
cat fruits.txt #display the content
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720186208779/a1f47565-0a1a-4daa-baae-4b7e35c43e10.png align="center")

## 7\. Show the first three fruits from the file in reverse order.

```bash
head -3 #filename | tac
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720186384579/5efd0557-6cd1-48e2-b4f4-a7b7e970d7c7.png align="center")

## 8\. Show the bottom three fruits from the file, and then sort them alphabetically.

```bash
tail -3 #filename | sort
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720186460968/b05a5b4f-bfe3-46e0-911c-40222e1e0d91.png align="center")

## 9\. Create another file `Colors.txt`, add content (one color per line), and display the content.

```bash
vim #file name
cat #file name
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720186756802/9fb030ce-48d8-4094-b431-894d4c6a2bd7.png align="center")

## 11\. Find and display the lines that are common between `fruits.txt` and `Colors.txt`.

```bash
comm -12 <(sort *filename1) <(sort *filename2)
```

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1720186940591/52fb80eb-14e9-4ce3-8448-53558bafd3e1.png align="center")

## 12\. Count the number of lines, words, and characters in both `fruits.txt` and `Colors.txt`.

```bash
wc #file name
```
